# Graphmint

Installable `src`-layout project for the `pilgrim` and `pilgrim_lightning`
packages.

## Install

Local editable install:

```bash
pip install -r requirements.txt
```

Direct install from Git:

```bash
pip install "graphmint @ git+https://github.com/Alice-mlops/Graphmint.git@main"
```

`cayleypy` is pinned in `pyproject.toml` to:

```text
git+https://github.com/cayleypy/cayleypy.git@e1518b6
```

## Packages

- `pilgrim`
- `pilgrim_lightning`
